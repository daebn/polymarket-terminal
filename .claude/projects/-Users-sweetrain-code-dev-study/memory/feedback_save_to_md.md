기획, 분석, 리서치 결과물은 항상 md 파일로 저장할 것.

**Why:** 유저가 토큰을 아끼고 싶어함. 대화 컨텍스트에 긴 내용을 반복 출력하지 않고 파일로 저장하면 나중에 Read로 참조 가능.

**How to apply:** 기획서, 분석 보고서, 리서치 결과 등 긴 출력물은 대화에 직접 쓰지 말고 프로젝트 디렉토리에 md 파일로 저장. 대화에서는 짧은 요약만 제공.
